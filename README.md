# Suncoast Retail Visual Analysis

## Overview

Suncoast Retail Visual Analysis is a Python-based data analytics and visualization project designed to simulate business performance across multiple Florida retail locations. The program generates realistic synthetic datasets for sales, advertising spend, customer demographics, and purchasing behavior, then uses visual analytics to uncover trends and insights. :contentReference[oaicite:0]{index=0}

This project demonstrates practical business intelligence skills such as trend analysis, market segmentation, comparative performance measurement, and dashboard creation.

---

## Features

### Sales Performance Analysis
- Quarterly sales trends
- Location-based sales comparisons
- Category performance by store
- Seasonal sales behavior

### Marketing Analytics
- Advertising spend vs sales relationship
- Return on advertising efficiency over time

### Customer Analytics
- Customer age distribution
- Purchase amount by age group
- Spending patterns by pricing tier

### Business Intelligence Dashboard
- Interactive multi-chart summary dashboard
- Key visual insights for management decisions
